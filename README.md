# Product D — Portfolio Ops Kit

An operations starter kit for maintaining portfolio visibility and execution hygiene.

## Includes
- Hourly audit script
- Hook runner + hook definitions (`hooks.toml` / `hooks.json`)
- Action queue and operation logs
- Ops docs for repeatable usage

## Target outcome
- Keep status fresh without manual drift
- Trigger lightweight follow-up actions automatically
- Preserve evidence for reviews and handoffs

## Scope note
Primary source files live under:
- `agent-org/scripts/`
- `agent-org/hooks/`
- `agent-org/reports/`
- `docs/ops/`
